Tentang Saya

Beberapa Keahlian Saya:

  • Pengembangan aplikasi desktop menggunakan C++.
  • Pengembangan website menggunakan HTML, CSS, dan JavaScript.
  • Pemrograman backend menggunakan PHP dan MySQL.
  • Pemahaman dasar tentang manajemen basis data dan SQL.

Tujuan saya adalah terus berkembang dalam bidang teknologi dan memberikan kontribusi yang berarti dalam proyek-proyek yang saya kerjakan. Saya percaya bahwa teknologi memiliki potensi besar untuk meningkatkan kehidupan kita, dan saya ingin menjadi bagian dari perkembangan itu.
